# 百年萨拉热窝 - 插图规划

> 当前所有视觉元素均使用 emoji，以下列出所有可替换为插图的位置，按优先级排列。

---

## 一、图片技术实现

项目已支持图片引用（`ui_dashboard.lua` 矿山横幅），用法：

```lua
UI.Panel {
    backgroundImage = "image/mine_banner.png",
    width = "100%", height = 160,
}
```

图片存放：`assets/image/`，引用时省略 `assets/` 前缀。

## 八、统一美术风格建议

- **色调**: 棕褐/暖色系（Sepia），呼应 1900-1945 时代感
- **画风**: 介于写实油画与游戏插画之间，略带颗粒/纸张质感
- **线条**: 科技/操作图标用统一粗细的铜色线稿
- **一致性**: 所有人物使用相同面部比例和光影方向（左上光源）
