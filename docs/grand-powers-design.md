## 1. 设计理念

### 1.1 核心比喻

> **欧洲是棋盘，大国是棋手，萨拉热窝是棋眼。**
>
> 大国 AI 在地图上推进、征服、撤退——这是"天气系统"。
> 玩家不能阻止暴风雨，但可以选择**造伞、造船、或者借风出海**。

### 1.2 三条核心原则

| 原则 | 含义 |
|------|------|
| **领土是真实的** | 地图上的国家会被 AI 实际征服/解放，颜色随主权变化 |
| **历史有惯性** | 事件大概率趋向史实，但玩家行为可以加速/延迟/偏移结果 |
| **小人物大杠杆** | 玩家是本地家族，不能直接指挥大军，但可以通过经济/情报/外交在关键节点撬动历史 |

### 1.3 与现有系统的关系

```
现有系统（保留）                 新增系统
┌─────────────────┐           ┌──────────────────┐
│ 3 个本地区域     │           │ 17 个欧洲国家     │
│ (矿区/工业城/首都)│◄────────►│ (领土控制+征服)   │
│                 │  影响      │                  │
│ 2 个本地AI      │           │ 4-5 个大国 AI     │
│ (家族/外资)      │◄────────►│ (军事/经济/外交)   │
│                 │  代理      │                  │
│ 43 个固定事件    │           │ 历史轨道+分支节点  │
│ (时间触发)       │◄────────►│ (条件触发)        │
└─────────────────┘           └──────────────────┘
```

本地 AI（家族势力/外资）变为大国在萨拉热窝的"代理人"，而非独立实体。

## 3. 大国 AI 实体

### 3.1 设计原则

大国 AI **不需要模拟完整的国家运行**（那是《维多利亚》的工作）。只需要：

1. **按历史轨道推进**（默认行为）
2. **根据玩家行为偏移**（被动反应）
3. **对萨拉热窝施加影响**（本地效果）

### 3.2 大国属性模型

```lua
Power = {
    id = "germany",
    name = "德意志帝国",
    active_years = {1907, 1945},    -- 活跃年份范围

    -- 三个核心指标（0-100，每季自动向历史基准漂移）
    military    = 70,    -- 军事力量（高→征服速度快）
    economy     = 80,    -- 经济实力（高→能支撑更多占领）
    war_fatigue = 0,     -- 厌战度（高→无法继续扩张，可能崩溃）

    -- 阵营标签（动态变化）
    faction = "central_powers",  -- central_powers / entente / axis / allies / comintern / neutral

    -- 征服目标队列（按优先级排序，由历史轨道预设）
    war_goals = {"france", "lowlands", "russia"},

    -- 对萨拉热窝的态度与政策
    attitude_to_player = 0,        -- -100 ~ +100
    local_policy = "neglect",      -- exploitation/cooperation/neglect/occupation
    local_proxy = "local_clan",    -- 在本地的代理势力（关联现有AI）
}
```

### 3.3 各时代活跃大国

场上始终保持 **3-4 个活跃大国**，按时代轮替：

| 时代 | 活跃大国 | 阵营对立 |
|------|---------|---------|
| Ch1 1907-1914 | 奥匈帝国, 德意志帝国, 俄罗斯帝国, (塞尔维亚) | 同盟国 vs 协约国 酝酿中 |
| Ch2 1914-1918 | **同盟国**(德+奥匈+奥斯曼) vs **协约国**(英+法+俄+意+塞) | 一战 |
| Ch3 1919-1938 | 南斯拉夫王国, 意大利, 纳粹德国 | 凡尔赛体系松动 |
| Ch4 1939-1945 | **轴心国**(德+意) vs **同盟国**(英+法+苏) + 游击队 | 二战 |
| Ch5 1946-1955 | 南斯拉夫(铁托), 苏联, 西方阵营 | 冷战 |

> Ch2 和 Ch4 是征服系统最活跃的阶段。Ch1/Ch3/Ch5 以外交/经济博弈为主。

### 3.4 大国的消亡与继承

| 事件 | 消亡的大国 | 继承/新生 | 领土变化 |
|------|----------|----------|---------|
| 一战结束 (1918) | 奥匈帝国, 奥斯曼帝国, 俄罗斯帝国 | 南斯拉夫王国(继承奥匈南部), 土耳其, 苏联 | 大量领土重划 |
| 纳粹崛起 (1933) | 魏玛共和国 | 纳粹德国(继承领土+扩张意图) | 德国 war_goals 激活 |
| 二战结束 (1945) | 纳粹德国, 法西斯意大利 | 占领区分治, 铁托南斯拉夫 | 恢复战前 + 铁幕 |

## 4. 领土征服系统

### 4.1 核心机制：前线推进

当两个阵营处于战争状态时，产生 **前线(front)**。前线沿邻接边界存在，每季度根据双方力量推进或后退。

```
征服流程:

大国A 对 国家B 发动攻击
    ↓
前线在 A-B 边界形成
    ↓
每季度计算: 攻击力 = A.military × (1 - A.war_fatigue/100)
           防御力 = B.stability × (B为大国 ? B.military : 30) / 100
    ↓
攻击力 > 防御力 → B 被征服: B.sovereign = A.id, B.resistance = B.stability
攻击力 ≤ 防御力 → 前线僵持: A.war_fatigue +5
```

### 4.2 征服不是一步完成

大国征服小国 = 1-2 季度自动完成（如德国闪击低地国家）。
大国对抗大国 = 需要多季度持续推进，受 `war_fatigue` 制约。

| 场景 | 预计耗时 | 历史对应 |
|------|---------|---------|
| 德国 → 低地国家 | 1 季度 | 1940年闪击荷比卢 |
| 德国 → 法国 | 2-3 季度 | 1940年法国战役 |
| 德国 → 俄罗斯 | 永远无法完全征服 | 东线拉锯 |
| 奥匈 → 塞尔维亚 | 3-4 季度 | 一战塞尔维亚战役 |

### 4.3 占领与抵抗

国家被征服后不会"消失"，而是进入**被占领状态**：

```lua
country = {
    sovereign = "germany",       -- 被德国控制
    original  = "france",        -- 但法国人民记得自己是法国
    resistance = 65,             -- 抵抗程度（来自原 stability）
    resistance_growth = 2,       -- 每季度抵抗+2
}
```

- 抵抗度 ≥ 80 → 可能爆发起义，占领方需要分兵镇压（military 下降）
- 抵抗度 ≥ 95 → 自动解放（占领方无力维持）

### 4.4 萨拉热窝的特殊位置

萨拉热窝（玩家所在地）的主权始终跟随 `austria_hungary`（或其继承国）的主权变化：

```
Ch1: 奥匈帝国领土 → 萨拉热窝由奥匈统治
Ch2: 战区（奥匈 vs 塞尔维亚前线）
Ch3: 南斯拉夫王国领土 → 萨拉热窝由南斯拉夫统治
Ch4: 被轴心国征服 → 萨拉热窝被占领
Ch5: 铁托南斯拉夫 → 萨拉热窝在联邦内
```

**当萨拉热窝被占领时**，玩家面临直接压迫：
- 占领方抽税（income × 占领税率）
- 经济管制（贸易受限）
- 征兵/强制劳动（人口下降）
- 但也有合作获利的机会

## 5. 玩家交互：四种姿态

玩家面对大国征服时，有四种基本姿态：

### 5.1 姿态总览

```
                    经济手段              军事手段
                ┌───────────┐        ┌───────────┐
  顺势而为      │  ① 合作    │        │  ② 加入   │
  (Join)       │  贸易/供给  │        │  派兵协助  │
                ├───────────┤        ├───────────┤
  逆势而动      │  ③ 制衡    │        │  ④ 抵抗   │
  (Resist)     │  制裁/走私  │        │  游击/破坏 │
                └───────────┘        └───────────┘
```

### 5.2 详细行动列表

#### ① 合作 (Collaborate) — 经济顺势

| 行动 | 条件 | 效果 | 风险 |
|------|------|------|------|
| 战争供给商 | 与交战大国 attitude ≥ 0 | 现金 +500-2000/季 | 对立方 attitude -10 |
| 矿产专供合同 | 拥有矿区控制 ≥ 50 | 矿石售价 ×1.5，但只能卖给该国 | 失去自由贸易权 |
| 合作经营 | 被占领时 | 收入稳定，家族安全 | 战后清算（声望-30, 罚没资产50%） |
| 引入资本 | 和平期 | 本地发展 +2，该国 interest +15 | 经济依赖性↑ |

#### ② 加入 (Join) — 军事顺势

| 行动 | 条件 | 效果 | 风险 |
|------|------|------|------|
| 派遣志愿军 | guards ≥ 10 | 该国 military +1，attitude +15 | 损失 2-5 guards/季 |
| 情报共享 | intel 层解锁 | 加速该国征服目标（-1季） | 被对方发现→attitude -30 |
| 镇压协助 | 被占领区 | 占领方 attitude +10，治安 +2 | 本地声望 -15，被抵抗组织报复 |
| 先发制人 | 与邻国关系恶化 | 对目标国 stability -10 | 其他大国态度恶化 |

#### ③ 制衡 (Counter) — 经济逆势

| 行动 | 条件 | 效果 | 风险 |
|------|------|------|------|
| 经济制裁 | influence ≥ 30 | 目标国 economy -3/季 | 贸易收入 -20% |
| 走私贸易 | 战时 | 向被封锁方出售物资，暴利 | 被抓→罚没 + 监禁（跳过2回合） |
| 资本外逃引导 | 金融知识 | 从不稳定国家吸引资本 | 目标国 attitude -15 |
| 货币战 | cash ≥ 8000 | 加速目标国 war_fatigue +3 | 自身 cash -2000 |

#### ④ 抵抗 (Resist) — 军事逆势

| 行动 | 条件 | 效果 | 风险 |
|------|------|------|------|
| 支持游击队 | 被占领时 | 占领国 war_fatigue +2，本地 resistance +5 | 被发现→家族损失重大 |
| 破坏补给线 | guards ≥ 5 | 占领国对该区域征服速度-1 | 损失 1-3 guards |
| 庇护难民 | 有空间 | 声望 +5，人口 +100 | 占领方 attitude -10 |
| 反推战线 | guards ≥ 20, 与解放方结盟 | 推动前线后退一步（改变一个国家主权） | 大量军事损耗 |

### 5.3 姿态的长期后果

玩家在整个战争期间的累计行为会被记录为 **合作度(collaboration_score)**：

```
collaboration_score:
  > +30  → 战后被定性为"合作者"，清算
    0-30 → "中间派"，无特殊后果
  < 0    → "抵抗者"，战后奖励
  < -30  → "英雄"，战后获得政治地位
```

这个分数直接影响 Ch4→Ch5 的过渡和战后事件选项。

## 6. 历史轨道与分支节点

### 6.1 历史轨道系统

每个大国的核心指标有一条 **历史基准线**——按季度预设的目标值。每季度，实际值向基准线**漂移**：

```lua
-- 每季度漂移逻辑
actual_value = actual_value + (baseline_value - actual_value) * DRIFT_RATE

-- DRIFT_RATE = 0.1 → 每季度缩小差距10%
-- 玩家行为会临时偏移 actual_value，但历史惯性会拉回来
-- 除非玩家在关键节点做出选择，永久改变基准线
```

**示例：德国军事力量基准线**
```
1907: 65  1910: 70  1913: 80  1914: 90(一战)  1917: 75(疲劳)  1918: 30(崩溃)
1919: 20  1925: 35  1933: 50(纳粹)  1938: 80  1939: 90(二战)  1943: 70  1945: 5(投降)
```

### 6.2 分支节点

**分支节点** = 玩家行为可以**永久改变**基准线的关键时刻。每章 1-2 个。

分支有三种可变性：

| 级别 | 含义 | 玩家影响 |
|------|------|---------|
| **可偏移** | 时间/烈度可变，但事件一定发生 | 推迟/加速 1-4 季度 |
| **可分支** | 可能走向不同结局 | 改变事件结果 |
| **蝴蝶效应** | 小行动引发连锁反应 | 改变后续整条基准线 |

## 7. 五章历史进程详述

### 7.1 第一章：铜版帝国 (1907-1914)

**大国格局**: 奥匈帝国统治萨拉热窝，塞尔维亚民族主义升温，德俄对峙

**领土变化**: 无（和平期），但张力持续积累

**大国行为**:
- 奥匈帝国: 对塞尔维亚施压，在波斯尼亚强化统治
- 塞尔维亚: 资助波斯尼亚民族主义运动
- 德国: 支持奥匈，对俄施压
- 俄罗斯: 支持塞尔维亚，泛斯拉夫主义

**玩家可做的事**:
- 从对立阵营之间的紧张关系中获利（两边卖物资）
- 与奥匈合作获得经济特权
- 暗中支持塞尔维亚民族运动（高风险高回报）
- 积累情报网络，为未来做准备

**🔀 分支节点：萨拉热窝刺杀 (1914 Q2)**

| 选择 | 条件 | 效果 |
|------|------|------|
| A. 向奥匈通报阴谋 | 与奥匈 attitude ≥ 20 | 刺杀失败，一战推迟 2-4 季度。奥匈 attitude +25，塞尔维亚视你为叛徒 |
| B. 协助刺杀组织 | 与塞尔维亚地下组织有联系 | 刺杀提前 1 季度，一战提前爆发。塞尔维亚 attitude +20 |
| C. 不干预 | 默认 | 按历史走向：1914 Q2 刺杀成功，Q3 一战爆发 |
| D. 刺杀后斡旋 | influence ≥ 40, 双方 attitude ≥ 0 | **蝴蝶效应**: 5%概率阻止战争升级，奥匈仅对塞尔维亚局部冲突（极难触发的隐藏结局） |

### 7.2 第二章：烽火铁幕 (1914-1918)

**大国格局**: 同盟国(德+奥匈+奥斯曼) vs 协约国(英+法+俄+意+塞)

**领土征服时间线（历史基准）**:

```
1914 Q3: 战争爆发，前线形成
1914 Q4: 德国 → 低地国家 (1季征服)
1915 Q1: 德国 → 法国北部 (前线僵持，未完全征服)
1915 Q2: 奥匈 → 塞尔维亚 (推进中)
1916 Q1: 塞尔维亚 陷落 (sovereign → austria_hungary)
1916 Q3: 罗马尼亚 加入协约国 → 被同盟国击败
1917 Q1: 俄罗斯革命 → stability 暴跌 → 退出战争
1917 Q3: 芬兰独立 (sovereign → 自身)
1918 Q2: 同盟国 war_fatigue 突破阈值
1918 Q3: 奥匈帝国崩溃 (stability → 0，解体)
1918 Q4: 德国投降，战争结束
```

**玩家可做的事**:
- **战争供给商**: 向任一阵营出售矿产/物资
- **走私贸易**: 绕过封锁线高价贩卖
- **庇护难民**: 接收塞尔维亚/比利时难民（人口+，声望+）
- **情报掮客**: 在两大阵营之间出售情报
- **前线影响**: 支持或破坏奥匈对塞尔维亚的攻势

**🔀 分支节点：奥匈帝国的命运 (1918)**

| 选择 | 条件 | 效果 |
|------|------|------|
| A. 推动联邦化改革 | 与奥匈 attitude ≥ 40, influence ≥ 50 | **可分支**: 奥匈未完全解体，改组为联邦（Ch3 政治格局完全不同） |
| B. 支持南斯拉夫统一 | 与塞尔维亚 attitude ≥ 30 | 默认结局加速，南斯拉夫更早成立 |
| C. 趁乱扩张地盘 | cash ≥ 5000 | 在权力真空中获取大量本地控制权，但政治风险高 |

### 7.3 第三章：镀金危机 (1919-1938)

**大国格局**: 凡尔赛体系下的脆弱和平。南斯拉夫王国统治萨拉热窝。意大利和德国不满现状。

**领土变化**: 
```
1919 Q1: 奥匈帝国解体 → 南斯拉夫王国成立 (继承波斯尼亚等)
1919 Q1: 奥斯曼帝国收缩 → 土耳其
1933 Q1: 纳粹德国上台 (war_goals 激活)
1936 Q1: 德国 → 莱茵兰重军事化 (military +10)
1938 Q1: 德国 → 奥地利合并 (austria 区域 sovereign → germany)
1938 Q3: 德国 → 要求苏台德地区
```

**这是和平积累期**，玩家应专注于：
- 经济建设，利用战后重建红利
- 与新成立的南斯拉夫建立关系
- 观察纳粹崛起，提前布局（囤物资、建情报网）
- 选择引入意大利/德国/英法资本

**🔀 分支节点：面对纳粹扩张 (1938)**

| 选择 | 条件 | 效果 |
|------|------|------|
| A. 与德国资本合作 | 经济导向 | 经济繁荣，但德国 interest +20（二战占领时被"特殊关照"） |
| B. 加固防御，备战 | guards ≥ 15 | 二战初期本地抵抗更强，占领推迟 1-2 季度 |
| C. 外交斡旋 | influence ≥ 60 | 尝试让南斯拉夫保持中立（**可分支**: 可能推迟被入侵） |

### 7.4 第四章：灰烬黎明 (1939-1945)

**大国格局**: 轴心国(德+意) vs 同盟国(英+法+苏) + 南斯拉夫游击队

**这是征服系统最活跃的阶段。**

**领土征服时间线（历史基准）**:
```
1939 Q3: 德国 → 波兰 (不在地图上，但标志战争开始)
1940 Q1: 德国 → 丹麦 (1季), → 低地国家 (1季)
1940 Q2: 德国 → 法国 (2季征服)
1940 Q3: 法国陷落 (sovereign → germany)
1941 Q1: 德国 → 南斯拉夫 (入侵开始)
1941 Q2: 南斯拉夫陷落 → 萨拉热窝被占领! (sovereign → germany/italy 分治)
1941 Q3: 德国 → 希腊 (1季)
1941 Q3: 德国 → 俄罗斯 (巴巴罗萨，长期拉锯)
1943 Q1: 斯大林格勒 → 德国 war_fatigue 飙升
1943 Q3: 意大利投降 → italy sovereign 恢复
1944 Q2: 诺曼底登陆 → 法国解放 (sovereign 恢复)
1944 Q3: 苏联反攻 → 罗马尼亚、保加利亚解放
1944 Q4: 南斯拉夫解放 → 萨拉热窝解放!
1945 Q2: 德国投降
```

**玩家在占领期（1941-1944）的核心选择**:

这不是单次事件选择，而是 **持续性的姿态管理**——每季度都在合作与抵抗之间权衡：

```
合作度 collaboration_score（持续追踪）

 +50 ─── 完全合作 ───── 经济稳定，但灵魂的代价
 +30 ─── 消极配合
   0 ─── 中间路线 ───── 表面配合，暗中准备
 -30 ─── 消极抵抗
 -50 ─── 积极抵抗 ───── 英雄之路，但可能全家覆灭
```

| 行动 | collaboration_score 变化 | 经济效果 | 军事效果 |
|------|--------------------------|---------|---------|
| 为占领方供应矿产 | +5/季 | cash +300 | — |
| 配合占领方征税 | +3/季 | 无额外损失 | — |
| 暗中隐藏资产 | -2/季 | 保住部分收入 | 被发现→罚没 |
| 资助游击队 | -8/季 | cash -200 | 占领方 war_fatigue +1 |
| 破坏铁路/补给 | -10/季 | — | 占领方征服速度-1 |
| 隐藏犹太难民 | -5/季 | — | 声望 +10 |

**🔀 分支节点：萨拉热窝解放方式 (1944)**

| 选择 | 条件 | 效果 |
|------|------|------|
| A. 等待苏联/游击队解放 | 默认 | 按历史走向，铁托游击队解放 |
| B. 组织本地起义 | collaboration ≤ -20, guards ≥ 15 | 提前 1-2 季解放，获得"解放者"称号 |
| C. 与西方盟军接洽 | influence ≥ 50 | **蝴蝶效应**: 萨拉热窝划入西方占领区（Ch5 完全不同！） |

### 7.5 第五章：战后余烬 (1946-1955)

**大国格局**: 铁托南斯拉夫 vs 苏联 vs 西方阵营（冷战）

**领土变化**: 铁幕形成，无军事征服，但阵营归属至关重要

```
1946: 东欧各国 sovereign 由苏联间接控制（卫星国）
1948: 铁托与斯大林决裂 → 南斯拉夫走第三条路
1949: NATO 成立
1955: 冷战格局稳定
```

**玩家的战后地位取决于 collaboration_score**:

| 合作度 | 战后待遇 |
|--------|---------|
| ≥ +30 | "合作者"清算：资产罚没 50%，被排除出公共生活 |
| +10 ~ +30 | "灰色地带"：小额罚款，保留大部分资产 |
| -10 ~ +10 | "普通市民"：无特殊影响 |
| ≤ -10 | "抵抗者"：获得政治地位，税收优惠 |
| ≤ -30 | "人民英雄"：免费获得国有化豁免，态度 +40 |

**🔀 分支节点：铁托的抉择 (1948)**

| 选择 | 条件 | 效果 |
|------|------|------|
| A. 支持铁托独立路线 | collaboration ≤ 0 | 西方援助提前，经济机会大 |
| B. 暗中为苏联传递情报 | 与苏联有渠道 | **可分支**: 决裂可能不发生，南斯拉夫成为卫星国（高难度结局） |
| C. 推动亲西方路线 | 解放时选了选项C | **蝴蝶效应**: 南斯拉夫加入西方阵营（极端替代历史） |

## 8. 与现有系统的整合

### 8.1 本地 AI 作为大国代理

现有的 `local_clan` 和 `foreign_capital` 不再是独立实体，而是大国在本地的**代理势力**：

| 时代 | local_clan 代表 | foreign_capital 代表 |
|------|----------------|---------------------|
| Ch1 | 波斯尼亚本地权贵 (亲奥匈) | 塞尔维亚民族主义者 |
| Ch2 | 奥匈军方代理 | 协约国秘密联络人 |
| Ch3 | 南斯拉夫政府代理 | 意大利/德国资本 |
| Ch4 | 占领军合作者 | 游击队地下组织 |
| Ch5 | 共产党地方干部 | 西方经济援助机构 |

**好处**: 不改变现有本地 AI 的数据结构和行为逻辑，只改变"谁在幕后"的叙事。大国系统通过调整本地 AI 的参数（现金、增长率、态度阈值）来体现大国的兴衰。

### 8.2 事件系统改造

现有 43 个固定事件中，**与大国直接相关的**改为由领土征服状态触发：

| 现有事件 | 现有触发 | 改造后触发 |
|---------|---------|----------|
| 一战爆发 (1914) | 固定年份 | 德俄张力 ≥ 85（通常在1914附近） |
| 战争结束 (1918) | 固定年份 | 同盟国所有大国 war_fatigue ≥ 90 |
| 德奥合并 (1938) | 固定年份 | 德国 military ≥ 75 且 economy ≥ 60 |
| 二战爆发 (1939) | 固定年份 | 德国 war_goals 队列激活 |
| 铁托决裂 (1948) | 固定年份 | 保留固定触发（冷战事件不需要征服系统） |

**其余事件保持固定触发不变**（经济危机、干旱等与大国征服无关的事件）。

### 8.3 对胜利条件的影响

胜利条件公式不变，但大国系统影响**难度曲线**：

| 大国状态 | 对玩家的影响 |
|---------|------------|
| 和平期 | 正常经济运行 |
| 萨拉热窝的宗主国在打仗 | 税率上升、贸易受限、征兵 |
| 萨拉热窝被占领 | 收入锐减、行动受限、但走私暴利 |
| 萨拉热窝刚解放 | 权力真空，快速扩张窗口 |
