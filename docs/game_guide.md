# 《萨拉热窝王朝》玩家攻略

> 你是萨拉热窝的望族家主，从1904到1955，在两次世界大战与冷战中经营矿业、壮大武装、左右逢源。

---

## 快速上手

- 每季度有固定行动点(AP)，做任何事都消耗AP，不够用时可以花钱买
- 核心循环：**赚钱 → 升级 → 扩张 → 应对战争**
- 胜利方式：积累足够的经济或军事胜利点后**手动宣告胜利**（1945年起可宣告）
- 留意通胀对现金的影响——想想什么资产不会贬值？

## 主面板

打开游戏默认显示的界面，一切操作的中枢。

### 事件流

- 每季开头会推送**待处理事件**（历史节点、随机事件等），标有"主线"或"支线"
- 点击"处理"进入事件选择，你的决定会影响合作度、资源、局势
- 战斗结果、AI动向、警告也会出现在这里

### 快速操作（4个按钮）

| 操作 | 说明 |
|------|------|
| 情报行动 | 侦查AI实力、渗透削弱、收买拉拢，费用和效果各不相同 |
| 科技研发 | 开启新科技研究，研发周期数季不等 |
| 政治外交 | 送礼改善关系、签订互不侵犯条约 |
| 资产交易 | 购买新矿山、出售矿山、突袭AI抢夺资金 |

> 每个操作点进去都能看到具体消耗和效果，按需选择即可。

### 焦点卡片

显示你的主要矿山状态：产出、利用率、工人士气、维护费，也可以快速招聘工人。

### 季节概览

- 储量条：黄金/白银/煤炭库存一目了然
- 经济指标：现金流、负债率、民心、影响力

### 结束回合

点击进入下一季。注意：
- 如果贷款利息付不起会警告，**连续违约4季会破产 Game Over**
- 净资产连续为负4季也会破产

### 买AP

顶栏有[+]按钮，花钱买额外AP，每季次数有限。

## 家族

管理你的家族成员和岗位分配。

### 成员属性

每位成员有5项属性（用圆点显示强弱）：管理、谋略、魅力、学识、野心。

### 岗位分配

共6个岗位，每个岗位看两项属性：

| 岗位 | 关键属性 | 作用方向 |
|------|---------|---------|
| 矿业总监 | 管理+学识 | 矿山产出 |
| 军务主管 | 谋略+野心 | 军事战力 |
| 财务总监 | 管理+谋略 | 经济效率 |
| 文化顾问 | 魅力+学识 | 影响力增长 |
| 科技顾问 | 学识+野心 | 研发速度 |
| 外交总监 | 魅力+谋略 | 外交效果 |

### 适配度

- **满配**（两项属性都很高）：获得完整加成
- **半配**（属性中等）：加成减半
- **差配**：反而有小额惩罚
- **空缺岗位**：该方向效率大幅下降，空缺太多还会扣AP

比较一下差配的惩罚和空缺的惩罚——哪个代价更大？

## 产业

矿山经营是你的经济命脉。

### 矿山运营

- 矿山最高 **5级**，每级提升产能。**升级费用随等级递增**（越高级越贵），且受通胀影响
- 矿山会慢慢耗尽。耗尽后三种处理方式：
  - **产能迁移**：花费AP和资金，保留部分产能
  - **善后处理**：花钱清理
  - **直接废弃**：不花钱但损失控制度

### 探矿

- 花费AP和资金启动，需要等待数季出结果
- 首次成功率尚可，之后逐次递减
- 最多持有 **2个备用矿脉**，探到后可激活

### 工人管理

- 可批量招聘/解雇工人
- 工人需要发工资，人越多产能越高
- 注意工人士气，工资低于区域均值时士气会降

### 黄金交易

- 可手动卖金（选择数量）
- 可开关**自动出售**：开了每季自动卖金换现金
- 想一想：战争来临时现金和黄金哪个更保值？自动出售开关的时机值得琢磨

### 收支概览

产业页顶部显示：季度收入、支出、净利润、白银/煤炭库存、通胀率。

## 市场

三个子标签页：股票、贷款、商品。

### 股票

- 6只股票可交易，每只有迷你走势图，显示涨跌趋势
- 点击可买入/卖出，输入数量即可交易
- 留意不同股票在和平期与战争期的表现差异，尤其是和军事相关的板块

### 贷款

三档贷款：小额短贷、中额贷款、大额长贷。期限越长利率越高。

需要注意：
- 实际利率会随你的**杠杆率**（总负债占资产比例）上浮——借得越多越贵
- 杠杆率过高时会被禁止再贷
- 同时最多持有 **3笔** 贷款
- 还不起利息会强制卖黄金、降级矿山；**连续违约4季破产**

不同档位风险差异很大，量力而行。

### 商品

- 显示黄金/白银当前价格和库存
- 通胀追踪：对比一下和平期与战时的通胀速度，思考这对你的资产配置意味着什么
- 自动交易说明

## 武装

管理你的护卫部队和军事力量。

### 护卫概览

- 显示：总人数、士气条、综合战力、每季军费
- 士气会自然衰减；打赢仗提升，打输大幅下降

### 兵力操作

- **招募**：花钱扩充护卫（受通胀影响）
- **裁撤**：缩减编制
- **补给**：补充消耗品，补给不足影响战力

### 编队管理

- 护卫必须**编入小队**才有完整战力
- 未编队的护卫战力大打折扣
- 编队面板可查看每个小队的装备、老兵等级、状态

### 装备系统

- 在兵工厂页面生产装备，从 T1（步枪）到 T6（精锐套件），等级越高战力越强
- 高等级装备需要科技解锁，生产周期更长、费用更高
- **注意装备耐久**：耐久降到约六成以下战力就开始打折，降到三成以下会严重衰减，归零则损毁。及时维修！

### 战斗规则

- **AI什么时候会打你**：态度很差（低于一定阈值）且兵力达标时，每季有概率主动进攻
- 你也可以主动突袭AI（消耗AP和资金）
- **打赢**：掠夺敌方部分现金，区域控制度上升
- **打输**：损失大量护卫，士气暴跌
- **提示**：注意观察AI态度和它进攻的关系——外交投入也许比军费更划算？

## 世界

三个子标签页：地图、势力、报告。

### 地图

- 互动地图展示3个区域（矿区/工业区/首都区）
- 点击区域查看详情：
  - **控制度**：你和各AI势力对该区域的控制占比
  - **资源**：该区域的金/银/煤/钢产出
  - **政治渗透**：花AP和影响力提升你的区域控制

### 势力

**大国势力**（一战/二战期间出现）：
- 显示军事力量、经济实力、战争疲劳度
- 你可以选择立场：合作/加入/对抗/抵抗
- 立场选择会影响合作度（详见下方"历史分支"）

**地方AI家族**：
- 显示态度、兵力、现金等信息
- 可进行外交操作（送礼、签约、宣战）

### 报告

- 季度总结：现金/黄金/影响力/护卫变化
- 全局指标：当前时代、战争状态、控制度、威胁等级、胜利点排名
- 历史日志：回顾过往大事件

## 跨模块要点

### 科技树（快速操作 → 科技研发）

四条路线，每条有**两处**分叉二选一，走了一边另一边就锁死：

- **采矿线**：提升矿山产出和探矿效率。分叉例如"通风系统"（稳定安全）vs"爆破开采"（暴力增产但事故多）
- **经济线**：加强贸易和金融。稳健选合法贸易路线，高风险选走私网络（收入高但影响力受损）
- **军事线**：强化战斗力。进攻选突击战术，防守选堑壕防御
- **文化线**：提升影响力和研发。大学加速研发+工人效率，宣传机器冲影响力+士气

> 科技选择没有绝对的对错，根据你当前的策略路线选择即可。

### 影响力

影响力每季自然衰减，需要持续做文化行动维持。达到特定里程碑会解锁强力被动效果：

- 达到一定影响力后会解锁被动效果，包括行动能力、研发速度、胜利点增速等
- 影响力容易衰减，需要持续投入文化行动来维持
- 具体里程碑数值和解锁效果可在「世界→势力」页面顶部查看

### 历史分支与合作度

游戏有 **5个关键历史节点**（1914刺杀事件、1918帝国崩溃、1938纳粹扩张、1944解放、1948铁托决裂），你的选择会累积「合作度」分数。

1946年战后清算时，合作度决定你的命运：

| 评定 | 后果 |
|------|------|
| **合作者**（合作度很高） | 资产大幅罚没，影响力减半——几乎是灾难 |
| **灰色地带**（合作度偏高） | 缴纳小额罚款，影响不大 |
| **普通市民**（中立） | 安然无事 |
| **抵抗者**（合作度偏低） | 获得现金奖励和声望 |
| **人民英雄**（合作度很低） | 巨额现金+黄金+影响力奖励，前途一片光明 |

> 不同的合作度路线各有利弊和风险。注意有些选项需要满足武装或影响力门槛才会出现。

## 各时期提示

### 1904-1913 和平期
- 和平时期是打基础的窗口——矿山、科技、影响力，哪个优先取决于你的胜利路线
- 关注兵工厂，装备升级需要时间
- 想想战争来临前，你的资产结构是否经得起通胀冲击？

### 1914-1918 一战
- 战时经济环境剧变，关注通胀对你各项资产的影响
- 战争状态下某些胜利点计算有加成，留意面板提示
- 外交手段在战时依然有用

### 1919-1940 战间期
- 历史上1929和1931年发生过重大经济事件，游戏中也会有对应冲击，做好准备
- 这段相对平静的时期适合升级和储备
- 思考你在下一场大战中要走什么路线

### 1941-1945 二战
- 二战的经济冲击比一战更剧烈
- 留意股市在战时的板块轮动
- 历史分支选择会在战后产生深远影响，做决定前看清后果

### 1946-1955 战后
- 战后清算的结果取决于你此前的选择，是危机也是机遇
- 1945年起可以宣告胜利，需要领先AI足够多
- 战后经济逐步稳定，是冲刺的好时机
