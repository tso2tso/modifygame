# 萨拉热窝王朝 · UI 设计规范文档
**版本 v1.0 · 面向 AI 辅助设计**

---

## 一、设计语言总纲

### 1.1 核心美学方向

**"工业帝国主义时代的家族账簿"**

视觉语言的核心隐喻是：一本19世纪末欧洲殖民地商人使用的厚重皮革账簿，内页是泛黄羊皮纸，数据用钢笔墨水刻印，边角有铜制扣件。

- **时代感**：1890–1910年代巴尔干半岛工业化初期，奥匈帝国殖民风格与东欧本土文化的张力
- **阶级感**：玩家是"拥有矿山、铁路、银行"的精英家族，UI 须传递沉甸甸的财富与压迫感
- **策略感**：每一个数字都有代价，界面不做任何多余装饰，信息密度高但有序

### 1.2 设计关键词

`历史感` · `信息密度` · `精英阶层` · `工业质感` · `紧张决策` · `手工雕刻`

### 1.3 严禁事项

- ❌ 禁止使用现代扁平风格（material design / iOS 默认控件感）
- ❌ 禁止使用渐变色背景（紫-蓝渐变、彩虹渐变）
- ❌ 禁止使用 Inter / Roboto / System UI 等现代无衬线字体作为主字体
- ❌ 禁止使用圆润过度的卡片（border-radius > 8px 的大卡片）
- ❌ 禁止使用高饱和度游戏风图标（如二次元风格、像素风）

## 二、色彩系统

### 2.1 主调色板

```
背景层
  --bg-base:        #1A1814   深煤炭棕黑（主背景）
  --bg-surface:     #242018   略浅的卡片底层背景
  --bg-elevated:    #2E2920   悬浮卡片/弹出面板背景

纸张层（内容卡片）
  --paper-dark:     #3D3426   深羊皮纸（卡片背景主色）
  --paper-mid:      #4A3E2E   中羊皮纸（内嵌区域）
  --paper-light:    #C8B89A   浅羊皮纸（分隔线、边框、次要文字）
  --paper-cream:    #E8D9C0   奶油白（主要内容文字背景参考）

文字层
  --text-primary:   #F0E6D0   主文本（近白奶油色）
  --text-secondary: #A89880   次要文本（灰棕色）
  --text-muted:     #6B5E4E   低优先级文本/占位符
  --text-label:     #C8B89A   标签/说明文字

强调色
  --accent-gold:    #C9A84C   金色（货币、高价值信息、高亮边框）
  --accent-red:     #C0392B   警戒红（危机事件、危险状态）
  --accent-amber:   #D4810A   琥珀橙（警告、中等风险）
  --accent-green:   #4A7C59   暗绿（正向数值、安全状态）
  --accent-blue:    #3A6B8A   钢蓝（信息类操作、外交）

功能色
  --danger:         #C0392B   
  --warning:        #D4810A   
  --success:        #4A7C59   
  --info:           #3A6B8A   
  --neutral:        #6B5E4E   
```

### 2.2 色彩使用规则

| 场景 | 使用色 | 说明 |
|------|--------|------|
| 主线事件标签 | `--accent-red` | 红色感叹号+文字 |
| 金币/现金数字 | `--accent-gold` | 所有货币数值 |
| 正增长数值 | `--accent-green` | 如 +3,520 |
| 负增长数值 | `--accent-red` | 如 -1,200 |
| AP 消耗提示 | `--text-secondary` | 灰色括号内文字 |
| 安全等级"低" | `--accent-red` + shield图标 | 带底色的状态胶囊 |
| 操作按钮悬停 | `--accent-gold` 边框 | 边框高亮，背景微亮 |
| 底部导航选中 | `--accent-gold` 下划线 | 图标+文字均高亮 |

### 2.3 纹理与图层

- **背景纹理**：主背景叠加 noise grain 纹理（opacity 3-5%），模拟老旧纸张质感
- **卡片边框**：1px solid `--paper-light`（opacity 40-60%），不使用阴影堆叠
- **分隔线**：0.5px `--paper-mid`，水平分割用
- **图片遮罩**：历史图片统一加 sepia 滤镜（100%）+ 低对比度处理，与整体色调融合

## 三、字体系统

### 3.1 字体选型

```
英文标题字体（品牌 / 地名 / 数字大标）
  font-family: 'Playfair Display', serif
  特点：高对比度衬线，19世纪报纸标题感
  
英文正文字体（按钮文字 / 标签 / 系统提示）
  font-family: 'EB Garamond', serif
  特点：优雅古典，适合长文本
  
中文字体（所有中文内容）
  font-family: 'Noto Serif SC', '思源宋体', serif
  特点：衬线宋体，与英文字体气质统一
  
数字专用（关键数据展示）
  font-family: 'Playfair Display', serif
  font-feature-settings: 'tnum' 1  /* 等宽数字 */
  特点：数字排列整齐，便于对比
```

### 3.2 字号规范

```
级别          字号    行高    字重    用途
──────────────────────────────────────────────
超大标题      28px   36px    700     弹出面板大标题
页面标题      22px   30px    600     顶栏年份/季节
卡片标题      18px   26px    600     矿场名称、系统名
副标题        15px   22px    500     分组标签
正文          13px   20px    400     事件描述正文
次要正文      12px   18px    400     说明文字、括号内容
标签          11px   16px    500     状态徽章、小标签
数据大字      32px   40px    700     关键KPI数字（现金/产量）
数据中字      20px   28px    600     次要数据（百分比/小计）
```

### 3.3 特殊文字处理

- **货币金额**：始终右对齐，使用等宽数字，前置货币符号（¥ 或 crown 符号）
- **负数**：红色 + 连字符，如 `-1,200`
- **正数增量**：绿色 + 加号，如 `+3,520`
- **百分比**：数字+`%`不分开，字号一致
- **剩余时间**：固定格式"剩余时间：X天"，红色倒计时感

## 四、组件规范库

### 4.1 顶栏（Top Bar）

```
布局：固定高度 72px，水平排列，背景 --bg-base
左侧：时间信息区
  - 年份+季节（22px Playfair Display，--text-primary）
  - 公历日期（12px，--text-secondary）
  
中部：核心资源 4 格（图标 + 数字）
  格式：[图标16px] [数值 16px bold] 
  资源顺序：💰现金 / 🟡黄金(吨) / ⚡产能 / ⭐声望
  分隔：1px 竖线 --paper-mid
  
右侧：⚙设置图标（24px，点击进入全局设置）

边框：底部 1px --paper-light（opacity 50%）
```

**状态变化**：
- 现金不足时，现金数字变为 `--accent-amber` 并轻微抖动动画
- 声望极低时（<30），声望数字变为 `--accent-red`

---

### 4.2 行动点区（AP Bar）

```
布局：顶栏下方独立行，高度 52px，背景 --bg-surface
左侧：[AP图标] [已用/总量] 文字，如 "4 / 6"
      字号 18px bold，--text-primary
      副文字"本季可用行动点" 11px --text-secondary
      
中部：点状进度条
  - 已用AP：实心圆点 ●（--accent-gold，直径 10px）
  - 未用AP：空心圆点 ○（--paper-light，直径 10px）
  - 超额AP（特殊状态）：闪烁金色点
  - 点间距：8px
  
右侧：[+] 按钮（获取额外AP，消耗其他资源）
      32px × 32px，圆角4px，--paper-dark 背景，--accent-gold 边框

右侧徽章：安全等级胶囊
  背景：--accent-red（低）/ --accent-amber（中）/ --accent-green（高）
  文字：白色 11px，"🛡 低 / 中 / 高"
  圆角：4px
```

---

### 4.3 事件流（Event Stream）

```
容器：--bg-surface 背景，内边距 16px
标题行：左"当前事件（N）" + 右"查看全部 ›"
        标题 15px 600，右侧链接 12px --accent-gold

事件卡片规范：
─────────────────────────────
│ [图片区]  │ [优先级徽章] 标题         │
│ 64×64px   │ 正文描述（2行 ellipsis）  │
│ sepia图   │ 剩余时间：X天             │
│            │                   [处理] │
─────────────────────────────
```

**优先级徽章**：
- `!主线`：`--accent-red` 实底，白字，圆角3px，12px
- `支线`：`--paper-mid` 底，`--text-secondary` 字
- 无徽章：普通信息类事件

**处理按钮**：
- 尺寸：72px × 32px
- 背景：`--paper-dark`，边框 `--paper-light`
- 悬停/点击：边框变 `--accent-gold`，背景微亮
- 文字：13px，`--text-primary`

**卡片间距**：`12px` 垂直间距，`0.5px --paper-mid` 分隔线

**紧急事件特殊样式**：
- 左侧添加 `3px solid --accent-red` 竖向边框线
- 标题文字变为 `--accent-red`
- 倒计时文字添加脉冲动画（opacity 1 → 0.6 → 1，1.5s loop）

### 4.4 经营焦点卡片（Focus Card）

这是主屏最重要的大卡片，展示当前最需关注的资产。

```
容器
  宽度：全宽，圆角 6px，边框 1px --paper-light（40%）
  背景：--paper-dark，内边距 16px

头部区
  左：资产名（18px 600，--text-primary）
  左：类型标签胶囊（如"金矿开采"，--bg-elevated 底，--paper-light 字，11px）
  右：更多操作 "…"

图片+核心KPI区（横向双栏）
  左：资产图片（历史感工业照片，sepia 100%，高度约 120px，圆角 4px）
  右：2×2 指标网格
      格式：[数值大字 22px bold --text-primary] + [单位 13px --text-secondary]
      标签：[12px --text-secondary]
      示例：产量(本季) 120吨 / 产能利用率 80%（附进度条）/ 工人状态 😊稳定 / 维护费用 -1,200

进度条规范（产能利用率等）
  高度：6px，圆角 3px
  底色：--paper-mid
  填充色：
    ≥80%：--accent-green
    50-79%：--accent-amber
    <50%：--accent-red

次要指标行（4格横排小卡片）
  每格：标题 11px + 数值 15px bold
  内容：库存 / 品质 / 开采成本 / 安全等级
  品质用文字（高/中/低），安全等级用带颜色徽章

操作按钮组（1AP/2AP 标注）
  横向 4 个按钮，等宽分布
  样式：高度 40px，圆角 4px，--bg-elevated 背景，1px --paper-light 边框
  文字：主文字 13px + 下方 (X AP) 11px --text-secondary
  悬停：--accent-gold 边框 + 背景微亮
  2AP 按钮：右上角小徽章标注"2"，--accent-amber 色
```

### 4.5 快速操作区（Quick Actions）

```
标题：15px 500，--text-secondary，"快速行动（消耗AP）"

按钮网格：3列×2行，或根据内容 6列×1行
每个按钮规范：
  尺寸：约 80px × 80px
  布局：垂直居中，图标（28px）+ 功能名（12px）+ AP消耗（11px --text-secondary）
  背景：--bg-elevated
  边框：1px --paper-mid
  圆角：6px
  图标风格：线性图标，--paper-light 色，非填充，笔触感（stroke-width 1.5）

点击效果：
  下沉感（scale 0.96 + 阴影内收）
  弹出对应底部操作面板（Drawer）
  
2AP 操作：按钮右上角显示"2AP"红色小圆角标签
```

| 金融操作 | 📈 折线+货币 | 1 |
| 科技研发 | 🔬 烧瓶/齿轮 | 1 |
| 情报行动 | 👁 眼睛/放大镜 | 1 |
| 政治外交 | 🤝 握手 | 2 |
| 资产交易 | 🏭 建筑/交易符号 | 2 |

### 4.8 右侧弹出面板（Side Drawer Panel）

弹出面板从右侧滑入，宽度约 220px（手机屏的 58%），遮住主屏右半部分。

```
容器：
  背景：--bg-elevated，左侧 1px --accent-gold 竖线（高亮边）
  圆角：左上+左下 8px
  内边距：16px

标题行：
  功能名（15px 600） + [×] 关闭按钮（右对齐）

  标题："可用成员"
  成员卡片列表：
    每卡：[X] 头像占位 + 姓名（13px）+ 属性（12px --text-secondary）+ [调配] 按钮
    [调配] 按钮：64px×28px，--paper-dark 底，--paper-light 边框
  底部："查看全部成员 ›" 链接

面板类型 2 - 金融操作：
  顶部：现金余额大字（24px bold --accent-gold）
  2×2 功能按钮网格：
    股票交易 / 贷款还款 / 发行债券 / 资金转移
    每格：图标（24px）+ 标签（12px），高度 64px，--bg-surface 背景，1px 边框
  底部：使用次数提示"本季已使用：X/Y次"（11px --text-muted）

面板类型 3 - 事件处理：
  事件标题（15px 600）
  副标题："选择你的应对方式："（12px --text-secondary）
  选项列表（单选）：
    每项：单选圆 + 主文字（13px）+ 副文字代价（12px --text-secondary）
    代价：绿色（正面）/ 红色（负面）/ 灰色（中性）
  [关闭] 按钮：全宽，--bg-elevated，--paper-light 边框，居中
  
触发方式：点击对应快速操作按钮 / 点击"处理"按钮
关闭方式：× 按钮 / 点击背景蒙层（蒙层 --bg-base opacity 60%）
动画：translateX(100%) → translateX(0)，ease-out 200ms
```

## 五、图标系统

### 5.1 图标风格定义

- **线性图标**（stroke-based），笔触宽度 1.5px
- **风格参考**：Phosphor Icons 中的 Regular 权重，或 Feather Icons
- **严禁**：emoji图标作为系统图标（除情感/状态表达外）
- **颜色**：跟随父元素文字色，不单独设置颜色

### 5.2 资源图标（顶栏专用）

顶栏资源图标需有时代感，建议使用定制 SVG：
- 💰 现金：古典金币图案（$符号+圆形）
- 🟡 黄金：金锭/矿石轮廓
- ⚡ 产能：蒸汽机/齿轮
- ⭐ 声望：皇冠/羽毛笔

### 5.3 状态图标

| 图标 | 含义 | 颜色 |
|------|------|------|
| 🛡 盾牌 | 安全等级 | 红/橙/绿 |
| ⚠ 警告三角 | 风险警示 | 琥珀色 |
| ❗感叹号 | 紧急事件 | 红色 |
| 😊 表情 | 工人状态满意 | 原色 |
| 😐 表情 | 工人状态普通 | --text-secondary |
| 😠 表情 | 工人状态不满 | --accent-red |

## 六、深度页面规范

从底部导航进入各深度系统页面，统一遵循以下框架：

### 6.1 深度页面框架

```
顶部：Sub-Header（56px）
  左：[←] 返回 / 页面标题（18px 600）
  右：可选的全局操作按钮
  背景：--bg-base，底部 1px --paper-mid

内容区：可滚动，背景 --bg-surface
  内边距：16px
  
底部：保留底部导航（同主屏）
```

### 6.2 家族页面

```
布局：树状家族结构图
  - 家主卡（中心）：头像+姓名+头衔，金色边框
  - 子女/亲属：向下排列，连线用 1px --paper-light 虚线
  - 职务角色标注：矿务总监/财务总监/军事主管 等，11px 徽章

家族成员列表（折叠区）：
  每人：头像 32px + 姓名 + 管理/学识/魅力 三维度评分
  评分用 1-10 数字 + 小点状图（●○ 形式）
```

### 6.3 产业页面

```
标签页筛选：全部 / 矿业 / 工业 / 运输 / 地产
  样式：下划线式标签，选中 --accent-gold

资产卡片列表：
  每卡：[资产图片 56×56] + [资产名 + 类型] + [产量 + 利润率] + [›] 进入
  产量/利润率：右对齐，颜色规则同前
  
底部：[+ 新增资产] 按钮（全宽，虚线边框，居中文字，--paper-mid 色）
```

### 6.4 市场页面

```
标签页：股票 / 债券 / 商品
  选中：下划线 --accent-gold + 文字高亮

股票列表规范：
  每行：[公司简称 15px] [现价 15px bold] [涨跌 13px 带颜色]
  交替背景：奇数行 --bg-surface，偶数行 --bg-elevated（微差）
  涨跌箭头：▲ --accent-green / ▼ --accent-red
  
底部：[我的持仓 ›] 按钮
```

### 6.5 武装页面

```
总兵力大字（顶部展示）：兵力数字 32px bold + 士气进度条

兵种分组：护矿队 / 巡逻队 / 游击队 / 预备队
  每组：标题 + 人数 + 驻扎地 + [详情] 按钮
  
底部：[武装单位详情] 跳转
```

### 6.6 世界页面

```
标签页：地图 / 关系 / 势力 / 报告

地图区：
  简化矢量地图（SVG），标注 3-4 个关键地点
  图钉样式：● 圆点 + 文字标注
  地点类型颜色：矿区（--accent-amber）/ 港口（--info）/ 城市（--text-primary）
  
关系面板：
  各势力关系值进度条（-100 ~ +100），居中对齐
  正值：--accent-green 填充
  负值：--accent-red 填充（从中心向左）
```

## 七、交互行为规范

### 7.1 核心交互模式

| 操作 | 触发方式 | 反馈 |
|------|---------|------|
| 消耗AP操作 | 点击按钮 | AP点减少动画（消失的圆点渐隐）|
| 打开弹出面板 | 点击快速操作 | 右侧 Drawer 滑入，背景蒙层 |
| 处理事件 | 点击"处理" | 弹出面板，显示选项 |
| 进入深度页 | 底部导航 | Push 动画（左滑进入）|
| 返回主屏 | ← 按钮 | Pop 动画（右滑返回）|
| 切换焦点卡片 | 横向滑动 | 卡片横向 carousel |

### 7.2 动画规范

```
所有 UI 动画遵循：快入慢出（ease-out）

标准时长：
  弹出面板滑入：200ms ease-out
  页面切换：280ms ease-in-out
  数字变化（资源增减）：400ms 数字滚动动画
  状态徽章出现：150ms scale(0.8→1) + fade in
  按钮点击反馈：80ms scale(1→0.96) 回弹

禁止：
  bounce 弹跳动画（与历史感不符）
  过于流畅的 spring 弹性（感觉轻浮）
  淡入淡出的模糊效果（过于现代）
```

### 7.3 加载状态

- **骨架屏**：用 `--paper-mid` 填充矩形占位，无闪烁动画（静态即可）
- **数据刷新**：右上角小加载指示器（旋转齿轮图标，16px）
- **操作中**：按钮文字变为 "处理中…"，禁用点击，透明度 60%

## 八、布局规范

### 8.1 移动端尺寸基准

```
目标设备：竖屏手机
参考宽度：390px（iPhone 14 Pro 逻辑像素）
最小支持：375px
内边距：全局水平 16px
安全区：顶部 statusBar（44px）+ 底部 homeIndicator（34px）
```

### 8.2 主屏层级结构（Z轴）

```
Layer 5（最高）: 弹出面板 Drawer + 蒙层
Layer 4:          模态对话框（确认操作）
Layer 3:          悬浮提示 Toast / Tooltip
Layer 2:          底部导航（fixed）
Layer 1（基础）:  主屏滚动内容
```

### 8.3 主屏内容滚动区域

主屏中间内容区可垂直滚动，以下区域固定不滚动：
- ✅ 固定：顶栏 + AP栏
- ✅ 固定：底部导航
- 🔄 滚动：事件流 + 焦点卡片 + 快速操作 + 本季概览
