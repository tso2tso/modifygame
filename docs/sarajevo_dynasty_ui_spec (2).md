# 《百年萨拉热窝：黄金家族》UI 设计规范文档 v1.1

> **文档用途**：本文档供 AI 辅助 UI 设计使用，描述界面结构、视觉语言、交互逻辑与时代适配规则。设计师与 AI 应将本文档作为首要参考，所有视觉决策须与文档一致。

> **v1.1 更新说明**：
> - **美术风格**：采用「写实插画背景 + 图标化 UI 管理」混合方案（以下简称「混合写实方案」）。纯图标方案信息效率更高但历史沉浸感弱；纯写实方案在竖屏小屏上信息密度不足。混合方案兼顾两者——背景、事件插画、成员肖像走写实，数据层、按钮、图标走扁平工业化，是最适合本游戏定位的路径。
> - **世界地图**：扩展为全欧洲范围节点地图，取代原有的局部节点图。
> - **地图图层**：采用文明 6 式分层叠加系统，详见第八章第 5 节「世界页」完整重写内容。

## 一、设计哲学

### 1.1 核心概念

本游戏是竖屏手机端的"百年家族经营模拟器"，UI 需要同时承载**策略信息密度**与**历史年代感**。设计方向是：

> **"帝国账簿 × 时代档案"**——界面既是一本工整的家族经营账本，又是一份承载百年历史痕迹的档案。随着时代推进，视觉语言会发生结构性演变。

### 1.2 三条设计铁律

1. **信息先于装饰**：所有装饰元素不得遮挡核心数值。每个像素的存在要么传递信息，要么服务于时代氛围。
2. **时代感必须可感知**：玩家不需要读文字，仅凭界面色调与纹理就能感受到"现在是什么年代"。
3. **单手拇指可达**：所有高频操作按钮（AP 消耗、处理事件、结束回合）必须落在屏幕下 60% 区域。

## 三、时代视觉系统（Era Visual System）

这是本游戏 UI 设计的核心创新。界面的色调、纹理、字体装饰、边框风格随时代章节切换，共分 7 套主题。

### 3.1 时代主题概览

#### 第一章（1904–1913）：铜版帝国
- **主色**：`#C8A96E`（旧金）、`#2C1810`（深棕）、`#F5EDD8`（羊皮纸）
- **强调色**：`#8B4513`（皮革棕）
- **背景纹理**：细粒羊皮纸纹理，轻微颗粒感
- **边框风格**：细线铜版雕刻风，转角带小装饰花纹
- **字体装饰**：标题使用衬线大写字母，数字用旧式数字体
- **UI 质感**：铜、皮革、木材

#### 第二章（1914–1918）：战报红章
- **主色**：`#1A1A1A`（战时黑）、`#8B0000`（警报红）、`#C8A96E`（褪色金）
- **强调色**：`#CC3300`（紧急红）
- **背景纹理**：旧报纸半调纹理，带轻微纸张破损感
- **边框风格**：粗线框，带有电报纸带装饰元素
- **特殊效果**：主线事件弹窗使用红色警报章印叠加
- **UI 质感**：铁、报纸、电报纸带

#### 第三章（1919–1940）：黑金工业
- **主色**：`#1C2331`（深蓝钢）、`#D4AF37`（纯金）、`#F0F0E8`（白色）
- **强调色**：`#4A7C59`（股票绿）、`#C0392B`（股票红）
- **背景纹理**：磨砂金属纹理，线条干净
- **边框风格**：Art Deco 几何线条，对称装饰
- **特殊效果**：股价翻牌数字动画，证券板风格
- **UI 质感**：铸铁、黄铜、玻璃

#### 第四章（1941–1945）：战时灰幕
- **主色**：`#2D2D2D`（战灰）、`#4A4A4A`（金属灰）、`#1A1A1A`（深黑）
- **强调色**：`#8B0000`（危机红）
- **背景纹理**：粗糙混凝土纹理，带划痕感
- **边框风格**：残破直线框，部分断裂效果
- **特殊效果**：低饱和度滤镜覆盖全局，警报时闪烁红边
- **UI 质感**：钢铁、混凝土、粗麻布

#### 第五章（1946–1991）：制度蓝红
- **主色**：`#1B3A6B`（政治蓝）、`#8B1A1A`（党政红）、`#F5F5F0`（表格白）
- **强调色**：`#2E7D32`（计划绿）
- **背景纹理**：细格子方格纸纹理，类似苏联风格表格
- **边框风格**：严谨直线框，无装饰，类制度化报表
- **特殊元素**：影子账户使用独立的暗色图层
- **UI 质感**：印刷品、搪瓷、铝合金

#### 第六章（1992–1995）：围城信号
- **主色**：`#3D3D3D`（围城灰）、`#6B6B6B`（混凝土）、`#1A1A1A`（黑夜）
- **强调色**：`#FF6600`（火光橙）、`#CC0000`（危机红）
- **背景纹理**：信号干扰纹理，扫描线效果
- **边框风格**：像素化、低分辨率感，偶发性界面"破损"特效
- **特殊效果**：收音机波纹动画，UI 偶发抖动
- **UI 质感**：废墟、应急设备、军用绿

#### 第七章（1996–2014）：玻璃金融
- **主色**：`#FFFFFF`（清洁白）、`#1C3557`（深海蓝）、`#E8F4F8`（浅蓝灰）
- **强调色**：`#00897B`（科技青）、`#F57C00`（数字橙）
- **背景纹理**：毛玻璃半透明效果，轻微阴影
- **边框风格**：细线、圆角、磨砂玻璃卡片
- **特殊效果**：新闻流、数据仪表盘风格
- **UI 质感**：玻璃、铝合金、屏幕光

### 3.2 时代过渡规则

- 时代切换时，界面进行 **1.5 秒渐变过渡动画**：主色淡出→新主色淡入
- 过渡期间显示年代更迭的全屏叙事卡片（历史背景文本 + 时代插画）
- AP 槽、资源栏等核心 HUD 在过渡完成后才重新显示

## 四、色彩设计系统

以下为跨时代通用的功能色，不随时代主题变化：

### 4.1 功能语义色

| 用途 | 颜色值 | 说明 |
|---|---|---|
| 正常 / 稳定 | `#4A7C59` | 绿色系，用于稳定状态、正收益 |
| 警告 / 风险 | `#E67E22` | 橙色系，用于中等风险、时限提醒 |
| 危机 / 主线 | `#C0392B` | 红色系，用于主线事件标签、紧急状态 |
| 信息 / 提示 | `#2980B9` | 蓝色系，用于一般提示、非紧急信息 |
| 禁用 / 耗尽 | `#95A5A6` | 灰色系，用于 AP 耗尽、不可操作状态 |
| 金币 / 奖励 | `#D4AF37` | 金色系，用于黄金、奖励、特殊成就 |

### 4.2 透明度规则

- 弹出面板背景：时代背景色 + `85%` 不透明度
- 禁用按钮：基础色 + `40%` 不透明度
- 悬停/选中高亮：白色叠加 `12%` 不透明度

## 五、字体系统

### 5.1 字体选用方向

游戏为中文界面，选用字体需兼顾历史感与可读性：

| 场景 | 字体风格建议 | 大小范围 |
|---|---|---|
| 标题 / 年份显示 | 衬线中文（方正悠黑 / 思源宋体 Heavy） | 20–28pt |
| 正文 / 说明文字 | 无衬线中文（苹方 / 思源黑体 Regular） | 12–14pt |
| 数值 / 资源数字 | 等宽数字字体（Tabular Figures） | 14–20pt |
| 事件标题 | 衬线加粗，略带年代装饰 | 15–17pt |
| 按钮标签 | 无衬线 Medium | 13–14pt |

### 5.2 字体层级

```
L1 页面标题：      20pt / Bold / 主色
L2 分区标题：      16pt / SemiBold / 主色 80%
L3 卡片标题：      14pt / Medium / 主色
L4 正文说明：      12pt / Regular / 主色 65%
L5 辅助标签：      11pt / Regular / 主色 45%
L6 数值强调：      18pt / Bold Tabular / 强调色
```

## 六、主界面（Main HUD）布局规范

主界面是玩家 80% 时间所在的页面，由 5 个固定区域组成：

```
┌─────────────────────────────┐
│         ① 顶栏信息区          │  高度：~88pt（含安全区）
├─────────────────────────────┤
│         ② 行动点条带          │  高度：44pt
├─────────────────────────────┤
│                             │
│       ③ 主内容滚动区          │  弹性高度（主体）
│    事件卡 + 经营焦点卡         │
│                             │
├─────────────────────────────┤
│       ④ 快速行动区            │  高度：80pt
├─────────────────────────────┤
│       ⑤ 底部导航 Tab         │  高度：83pt（含安全区）
└─────────────────────────────┘
```

### 6.1 ① 顶栏信息区

**布局**：单行，横向排列所有核心全局资源

```
[年份·季节]  [现金]  [黄金]  [产能]  [声望]  [⚙设置]
```

**规格**：
- 背景：时代主色，带底部细线分隔
- 年份+季节：左对齐，显示如 `1904年 Q2 春季` 和子标题 `4月15日`，L1 字级
- 资源图标 + 数值：图标 18×18pt，数值 L6 字级，间距均分
- 若资源变化（结算后），数字做 **向上滚动计数动画**（0.4秒）
- 声望（Reputation）显示为 0–100 的整数，波动时带 +/- 浮动提示
- 设置图标：右对齐，固定位置

**特殊处理**：
- 当处于战争状态（第二/四/六章），顶栏背景叠加半透明红色条纹
- 当安全等级为"低"时，声望图标旁显示红色警示盾牌

### 6.2 ② 行动点（AP）条带

AP 是核心决策资源，需始终清晰可见。

**布局**：

```
[AP图标] 4 / 6 本季可用行动点   ●●●●○○   [+]
                                         安全等级：[🛡低]
```

**规格**：
- AP 点用 **实心圆●** 表示已用，**空心圆○** 表示剩余
- 最大 AP（含加成）上限 10，以圆点组排列，超过 6 个时换排
- 已用 AP（●）：时代主色
- 剩余 AP（○）：时代主色 30% 透明度
- `[+]` 按钮：临时 AP 奖励时出现，点击后弹出说明
- 安全等级标签：右侧独立显示，颜色参照功能色（高→绿，中→橙，低→红）

### 6.3 ③ 主内容滚动区

可竖向滚动，包含两种主要内容组：

#### A. 事件卡列表（优先级最高，置顶）

标题：`当前事件 (N)` + 右侧 `查看全部 >`

事件卡规格：
- 宽度：全宽（边距 16pt）
- 高度：动态（最低 88pt）
- 左侧：96×64pt 事件插画（灰底占位图，带年代感滤镜）
- 上方：事件优先级标签（主线=红色`!主线`、一般=橙色、随机=灰色）
- 标题：L3 字级 Bold
- 描述：L4 字级，最多 3 行，超出省略
- 剩余时间：L5 字级，`剩余时间：N天`，临近时变为红色
- 右下：`处理` 主按钮（CTA 样式，高 36pt，宽 72pt）

事件卡间距：8pt

#### B. 经营焦点卡（当前主要资产）

标题：资产名称 + 类型标签

**焦点卡规格**（以金矿为例）：
- 顶部：资产名称（L2）+ 类型标签（如`金矿开采`，L5 的 Badge 样式）
- 左侧配图：资产插画，148×100pt，带时代感处理
- 右侧指标网格（2列）：
  - 产量（本季）、产能利用率
  - 工人状态（表情图标+文字）、维护费用（红色负数）
- 底部指标行（4格）：库存、品质、开采成本、安全等级
- 操作按钮行：横排 4 个操作按钮，L5 标签，注明 AP 消耗

**指标格规格**：
- 图标 16×16pt + 标签（L5）+ 数值（L3 Bold）
- 危险数值（负数、低等级）用危机红显示
- 每格最小宽度 80pt

### 6.4 ④ 快速行动区

标题：`快速行动（消耗AP）`

6 个行动类别图标按横向排列（可滑动），每个图标块：
- 图标容器：56×56pt，圆角 12pt，时代主色背景
- 图标：白色线条图标，28×28pt
- 标签：L5 字级
- AP 消耗标签：`1 AP` / `2 AP`，L5 字级

点击任一块 → 从底部弹出对应操作面板（Bottom Sheet）

**6 个快速行动**（按 GDD 中的行动分类）：

| 图标标识 | 标签 | AP |
|---|---|---|
| 👥 人像 | 人事管理 | 1 AP |
| 📈 图表 | 金融操作 | 1 AP |
| 🔬 烧瓶 | 科技研发 | 1 AP |
| 👁 眼睛 | 情报行动 | 1 AP |
| 🤝 握手 | 政治外交 | 2 AP |
| 🏭 建筑 | 资产交易 | 2 AP |

**AP 不足时**：相关按钮变为禁用态（灰色 + 锁图标），不可点击。

### 6.5 ⑤ 底部导航 Tab

5 个固定 Tab，横向均分：

```
[家族]  [产业]  [市场]  [武装]  [世界]
```

- 图标：20×20pt 线条图标
- 标签：L5 字级
- 激活状态：图标实心 + 标签加粗 + 底部 2pt 亮色指示条
- 未激活：图标线条 + 时代主色 50% 透明度

## 七、右侧弹出面板（Flyout Panels）

点击快速行动区的图标后，从**屏幕右侧滑入**一个宽约 240pt 的抽屉面板。

**通用规格**：
- 宽度：屏幕宽度 × 62%（约 240pt）
- 背景：时代主色 92% 不透明 + 高斯模糊背景
- 顶部：面板标题 + ✕ 关闭按钮
- 进入动画：右侧滑入，0.25s ease-out
- 关闭：点击空白区域或 ✕

### 7.1 人事管理面板

- 当前可用成员列表（家族成员 + 可雇用员工）
- 每个条目：头像 40×40pt（圆形）+ 姓名 + 关键属性 2 项
- 右侧：`调配` 按钮（次要样式，高 32pt）
- 底部：`查看全部成员 >` 链接

### 7.2 金融操作面板

- 顶部：当前现金显示（L1 字级，带货币符号）
- 4 个功能块（2×2 网格）：
  - 股票交易、贷款/还款
  - 发行债券、资金转移
- 每块：图标 + 标签，点击进入对应子页
- 底部：`本季已使用：N / 2 次`（AP 上限提示）

### 7.3 事件处理面板

- 事件标题（L2 Bold）
- 选项列表（通常 3–4 项）：
  - 每项：左侧标记○ + 选项文字 + AP 消耗小标签 + 效果预览
  - 隐藏选项（路线专属）：用金色边框高亮，并标注解锁条件
- `关闭` 次要按钮 / `确认执行` 主按钮

## 八、各子页面规范

### 8.1 家族页（Family）

**用途**：管理家族核心成员（最多6人）的属性与岗位分配

**页面结构**：

```
[页头：家族名称 + 家族总实力评级]
─────────────────────────────
[家族树结构图]（上方简要树状图，最多3层）
─────────────────────────────
[成员卡片滚动列表]
─────────────────────────────
[家族成员列表入口 > （完整列表含员工）]
```

**家族树规格**：
- 树节点：60×60pt 圆形头像卡 + 角色名（L5）
- 连线：时代主色细线
- 岗位显示：每个节点右下角 Badge 显示当前岗位

**成员详情卡（展开态）**：
- 左：头像（80×80pt，方形圆角卡片，带年代感框）
- 右：姓名（L2）、岗位（L5 Badge）
- 五项属性条：管理/谋略/魅力/学识/野心
  - 数值 1–10，用分段进度条表示（10格）
  - 满配岗位时对应属性条变金色高亮
- 底部：隐藏倾向线索（不显示数值，只显示事件触发后的文字暗示）
- 操作：`调整岗位（1 AP）` 按钮

**岗位空缺处理**：
- 空缺岗位用虚线边框卡片表示，内有 `+` 图标 + `空缺` 文字
- 空缺导致的 AP 惩罚在顶部 AP 条旁用⚠图标提示

### 8.2 产业页（Industry）

**用途**：管理所有资产（矿山/工厂/运输）的生产状态

**页面结构**：

```
[筛选 Tab：全部 | 矿业 | 工业 | 运输 | 地产]
─────────────────────────────
[资产列表（竖向滚动）]
─────────────────────────────
[+ 新建资产] 底部固定按钮
```

**资产列表条目**：
- 高度：72pt
- 左：资产图标（40×40pt）
- 中：资产名称（L3）+ 状态标签（运营中/停产/低效）
- 右：产能利用率百分比（L3 Bold，按状态着色）+ 利润率（L4）
- 点击 → 展开详情或跳转资产详情页

**资产详情子页**：
- 和主界面焦点卡类似的完整数据展示
- 但包含完整历史产量图（简单折线图，按季度）
- 操作按钮：调整生产、雇佣工人、武装护卫、升级设施（各注明 AP 消耗）

**第五章专属**（1946–1991）：
- 每个资产条目右侧显示 **配额状态**（已达配额/超额/未达）

#### 地图子页（核心，完整规范）

世界地图是本页核心，覆盖**全欧洲范围**，从伊比利亚半岛到乌拉尔山，从斯堪的纳维亚到北非海岸线，以萨拉热窝/波黑为默认视觉中心。

##### 地图基础

**地图类型**：矢量化抽象政治地图，非写实卫星地图。风格参考文明 6 的六边形区域感，但本游戏采用**多边形区域（省/国家轮廓）+ 离散节点**的混合结构。

**地图层级**：
- **宏观层**（默认视角）：可见全欧洲，国家/大区轮廓清晰，节点为小圆点
- **区域层**（双指放大 或 点击某国进入）：可见该国内各省/城市节点，路线细节显示
- 两层之间通过**缩放手势**切换，过渡动画 0.3s ease-out

**地图画布规格**：
- 渲染区域：全屏（顶部 Tab 栏下方至底部 Tab 上方），约 390×650pt
- 支持双指缩放（最小 0.5×，最大 4×）和单指平移
- 默认视图：全欧洲铺满屏幕，波黑区域用轻微高亮框标注「家族根据地」

**地图底图风格**（随时代主题变化）：

| 时代 | 底图风格 |
|---|---|
| 1904–1918 | 旧地图手绘风，羊皮纸底色，国境线为油墨手绘细线 |
| 1919–1940 | 印刷色块政治地图，Art Deco 边框 |
| 1941–1945 | 军事作战地图风格，等高线纹理，标注前线 |
| 1946–1991 | 冷战阵营地图，铁幕用粗虚线分隔东西方 |
| 1992–1995 | 低饱和冲突地图，热点区域用橙红晕染 |
| 1996–2014 | 现代卫星感地图，清洁线条，EU 范围浅蓝色 |

##### 图层系统（Civ6 式分层叠加）

地图顶部有**图层选择器**，横向排列，可同时激活多个图层叠加显示（最多 2 个叠加，避免信息过载）。

**图层选择器 UI**：
- 位置：地图顶部悬浮条，半透明背景，圆角胶囊形
- 每个图层按钮：文字标签 + 对应颜色方块，高度 32pt
- 激活状态：按钮填充对应图层色，文字白色
- 非激活：透明底，文字灰色

**可选图层清单**：

| 图层名 | 图层代号 | 视觉表现 | 核心信息 |
|---|---|---|---|
| 控制范围 | CONTROL | 区域色块填充（玩家蓝/AI各色/中立灰） | 各势力控制的节点与区域 |
| 资源分布 | RESOURCE | 节点上叠加资源图标（金/煤/铁/粮） | 每个节点的核心资源类型与储量 |
| 安全等级 | SECURITY | 区域热力图（绿→黄→红渐变色） | 各地区治安状况 |
| 贸易路线 | TRADE | 流动线条（动态虚线，方向动画） | 主要运输/贸易路线及其效率 |
| 政治影响 | POLITICS | 辐射状渐变圆（影响力由深到浅） | 家族与各势力的政治渗透范围 |
| 军事部署 | MILITARY | 军旗图标 + 势力范围圆圈 | 各武装单位位置与地区控制 |
| 文化影响 | CULTURE | 暖色光晕（金/橙，从文化资产向外辐射） | 文化产业覆盖与声望扩散范围 |
| 情报覆盖 | INTEL | 「迷雾战争」叠加（已知区域清晰，未知区域加半透明迷雾） | 当前情报等级覆盖的可见范围 |

**图层叠加规则**：
- 底图 + 控制范围图层始终作为基础显示（不可关闭）
- 其余图层最多同时激活 2 个
- 「情报覆盖」图层激活时，迷雾叠加在所有其他图层之上
- 军事图层仅在玩家拥有武装单位后解锁

**图层间视觉优先级**（从下到上）：
```
底图 → 控制色块 → 资源/政治/文化（面图层）→ 贸易路线（线图层）→ 军事/情报（点/迷雾图层）→ 节点图标 → 交互高亮
```

## 九、事件系统 UI 规范

事件是游戏最高频的叙事界面，需要最精细的设计。

### 9.1 事件弹窗（Modal）规范

触发事件时，弹出全屏半遮罩 Modal，中心显示事件卡。

**事件卡结构（从上到下）**：

```
┌──────────────────────────┐
│  [事件类型 Badge]  [优先级] │
│                          │
│  [事件插画 / 场景图]       │  高度约 160pt
│  390×160pt，带时代感滤镜   │
│                          │
│  [事件标题]               │  L1 Bold
│  [日期 · 事件分类]         │  L5
│                          │
│  [事件描述文本]            │  L4，最多 6 行
│                          │
│  ──── 选择你的应对方式 ──── │  L5 分隔线
│                          │
│  ○ [选项一文本]           │
│    效果预览：+现金 -风险   │  L5 灰色
│                          │
│  ○ [选项二文本]           │
│    效果预览：...           │
│                          │
│  ◆ [专属路线选项]          │  金色边框，路线解锁
│    解锁条件：文化影响 ≥20   │
│                          │
│  [确认执行]               │  主 CTA 按钮
└──────────────────────────┘
```

### 9.2 事件优先级视觉差异

| 优先级 | Badge 样式 | 插画风格 | 边框 |
|---|---|---|---|
| 主线历史事件 | `! 主线`，红色填充 | 全彩年代插画 | 金色双线 |
| 地区事件 | `地区`，橙色填充 | 半色调插画 | 时代主色单线 |
| 随机事件 | `随机`，灰色填充 | 图标化插画 | 灰色细线 |
| 家族事件 | `家族`，紫色填充 | 成员肖像 | 紫色单线 |

### 9.3 事件效果预览规则

- 正面效果：绿色文字 + ▲
- 负面效果：红色文字 + ▼
- 中性/未知效果：灰色文字 + ?
- 隐藏效果（高风险选项）：用 `···` 表示部分未知后果

## 十二、动效规范

### 12.1 基础动效参数

| 场景 | 时长 | 缓动 |
|---|---|---|
| 页面切换（底部 Tab） | 200ms | ease-in-out |
| 弹出面板（右侧滑入） | 250ms | ease-out |
| 事件 Modal 出现 | 300ms | spring（弹性） |
| AP 点更新 | 350ms | ease-in-out |
| 数值计数变化 | 400ms | linear |
| 时代切换过渡 | 1500ms | ease-in-out 双阶段 |
| 卡片展开 | 200ms | ease-out |

### 12.2 特殊动效

**资源结算动画**（每季回合结算时）：
- 各资源数值同时滚动更新
- 增加的资源：金色粒子上升动效 0.5s
- 减少的资源：红色闪烁 1 次后淡出

**AP 消耗动画**：
- 点击消耗 AP 的操作后，对应 AP 圆点从实心→空心，0.2s 缩小后变色

**主线事件触发**：
- 全屏黑色蒙版淡入 0.3s
- 事件标题文字从屏幕中心放大显现
- 配合音效（铃声/广播声/枪声，按时代）

**股价变化**（第三章后）：
- 数字翻牌动效（Slot Machine 风格，每位数字独立滚动）
- 涨：绿色，向上动效；跌：红色，向下动效

**围城章节干扰特效**（第六章）：
- UI 偶发水平位移 2–4pt，持续 0.1s，每 30–90s 随机触发
- 类信号干扰的扫描线叠加（低不透明度）

## 十三、信息层级与视觉重量控制

主界面同时承载多层信息，需严格控制视觉焦点：

### 13.1 注意力优先级

```
P1（必须立即处理）：主线事件红色 Badge + Modal 遮罩
P2（本季重要）：     AP 计数器 + 待处理事件数
P3（经营核心）：     资源数值 + 焦点卡数据
P4（辅助决策）：     快速行动按钮 + 底栏导航
P5（背景信息）：     本季概览条带 + 状态标签
```

### 13.2 信息密度控制原则

- 同一屏幕可见区域内，L1 字级元素不超过 3 个
- 红色（危机色）元素同时可见不超过 2 处
- 每张卡片最多暴露 4 个数值，其余折叠进详情

## 十四、空状态与加载状态

**空状态**（如未持有任何股票）：

- 图标使用时代风格的装饰性图案

**加载状态**：

  - 1–3章：铜版蚀刻风格旋转齿轮
  - 4–5章：转动的打字机字轮
  - 6章：雷达扫描圆圈
  - 7章：数据流进度环

## 十五、特殊功能界面

### 15.1 结束回合确认

- 仅在无待处理事件时激活
- 底部显示浮动横条：`[▶▶ 跳过本季，按上季度指令执行]`
- 点击后显示预计结算摘要，二次确认后执行

### 15.3 情报预警按钮

- 固定显示在主界面左下角（浮动，不随 Tab 变化）
- 图标：眼睛形状 + 情报等级小数字 Badge
- 点击 → 底部弹出情报面板，显示：
  - 当前情报等级（1–3）及衰减倒计时
